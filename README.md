# CustomWxpython

<img width="300" height="300" alt="image" src="cwx/assets/icon.png" />

Wxpython在MFC上十分甚至九分难看, 这就是本库的由来

CustomWxpython是一个基于Wxpython的组件库, 用于组建现代的、美观的GUI界面, 并提供暗色模式！
库的设计及样式高度基于WinUI 3
